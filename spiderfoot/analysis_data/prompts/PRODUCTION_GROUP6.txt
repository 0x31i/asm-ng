CONFIGURATION:

- Mode: PRODUCTION (analyze ALL rows)
- Output: ./output/
- Data: TOKENIZED/OBFUSCATED (IPs, domains, hosts are token identifiers)

CONSOLIDATION RULES (MANDATORY):
The goal is a concise, executive-readable report. Minimize finding count through
aggressive consolidation while preserving detail in supporting CSVs.
- NEVER create one finding per individual port, IP, domain, URL, or asset instance.
- Group related findings by EVENT TYPE + SEVERITY TIER.
- The consolidation phase groups by (Category, Vuln_Type, Priority) so multiple
  instances of the same event type at the same severity merge into a single finding
  with instance count and asset list.
- Detail CSVs capture every individual instance for drill-down.
- Target: under 20 consolidated findings per group. If consolidation produces more
  than 20, review Vuln_Type keys for over-granularity.

```python
def normalize_priority(severity, default='Medium'):
    """Normalize severity to standard format"""
    if not severity or str(severity).lower().strip() in ['nan', '', 'none']:
        return default
    severity = str(severity).strip().lower()
    if severity in ['critical', 'crit', '4']:
        return 'Critical'
    elif severity in ['high', 'hi', '3']:
        return 'High'
    elif severity in ['medium', 'med', '2']:
        return 'Medium'
    elif severity in ['low', 'lo', '1']:
        return 'Low'
    elif severity in ['info', 'informational', '0']:
        return 'Info'
    return default

def get_cvr_scores(row):
    """Extract CVR scores from a row (10-col format only)."""
    if CSV_FORMAT != 'full_scored':
        return None, None, None
    try:
        confidence = int(row.get('Confidence', 0) or 0)
        visibility = int(row.get('Visibility', 0) or 0)
        risk = int(row.get('Risk', 0) or 0)
        return confidence, visibility, risk
    except (ValueError, TypeError):
        return 0, 0, 0

def get_tracking(row):
    """Extract tracking status from row."""
    return str(row.get('Tracking', 'OPEN')).strip().upper() or 'OPEN'

def get_fp(row):
    """Extract FP value. Returns int: 0=unverified, 1=false positive, 2=verified."""
    try:
        return int(row.get('F/P', 0) or 0)
    except (ValueError, TypeError):
        return 0

def parse_severity_from_data(data_field, default='Medium'):
    """Parse severity level from the Data field text."""
    data_lower = str(data_field).lower()
    if 'critical' in data_lower:
        return 'Critical'
    elif 'high' in data_lower:
        return 'High'
    elif 'medium' in data_lower or 'med' in data_lower:
        return 'Medium'
    elif 'low' in data_lower:
        return 'Low'
    elif 'info' in data_lower or 'informational' in data_lower:
        return 'Info'
    return default
```

ASM-NG GRADE SCORING REFERENCE (from grade_config.py):
Category: DNS Health | Weight: 0.7
Score Formula: Score = max(0, 100 + (raw_score * 0.7))
Grade Thresholds: A=90+, B=80+, C=70+, D=60+, F=<60

EVENT TYPE SCORING:
| Event Type | Rank | Points | Logic |
|------------|------|--------|-------|
| DNS_SPF | 2 | -10 | zero_entries_fail (fail_points: -50) |
| DNS_TEXT | 2 | -10 | zero_entries_fail (fail_points: -50) |
| DNS_SRV | 3 | 0 | informational |
| DOMAIN_IMPERSONATION | 5 | 0 | informational |

IMPORTANT - zero_entries_fail LOGIC:
- DNS_SPF and DNS_TEXT use zero_entries_fail scoring
- If NO rows found for DNS_SPF -> Critical finding (SPF not configured, -50pts)
- If rows found for DNS_SPF -> analyze the SPF records for misconfiguration (Medium)
- Same logic for DNS_TEXT
- This is DIFFERENT from all other event types!

```python
print("\n=== Processing DNS Security Records (zero_entries_fail logic) ===")

for dns_type, record_name, fail_description in [
    ('DNS_SPF', 'SPF (Sender Policy Framework)', 'SPF record not configured - email spoofing protection missing'),
    ('DNS_TEXT', 'TXT (DNS Text Records)', 'DNS TXT records not configured - missing email authentication and domain verification'),
]:
    rows = ROWS_BY_TYPE.get(dns_type, [])
    print(f"\n  Processing {dns_type} ({len(rows)} rows)")

    tab_findings = []

    # ZERO ENTRIES FAIL CHECK: If no rows exist, this is a Critical finding
    if not rows:
        print(f"  ZERO_ENTRIES_FAIL TRIGGERED: No {dns_type} rows found!")
        group_stats['zero_entries_fail_triggered'] += 1

        severity = 'Critical'

        if dns_type == 'DNS_SPF':
            description = f"No SPF (Sender Policy Framework) records were found for any domain in the attack surface. "
            description += "SPF records are essential email authentication mechanisms that specify which mail servers are authorized to send email on behalf of a domain. "
            description += "Without SPF, attackers can freely spoof emails from the organization's domain, enabling phishing, business email compromise, and social engineering attacks. "
            description += "The absence of SPF records is a critical email security gap that directly enables domain spoofing and should be remediated immediately. "
            description += "This finding carries a penalty of -50 points due to the zero_entries_fail scoring logic."

            recommendation = "Immediately configure SPF records for all organizational domains by publishing TXT records in DNS specifying authorized sending IP addresses and mail servers. "
            recommendation += "Start with a monitoring policy (v=spf1 ... ~all) and transition to a strict enforcement policy (v=spf1 ... -all) after verifying all legitimate senders. "
            recommendation += "Complement SPF with DKIM and DMARC records to establish comprehensive email authentication. "
            recommendation += "Monitor SPF alignment reports to identify unauthorized senders and refine the policy. "
            recommendation += "Priority: Critical - configure within 7 days to prevent email spoofing attacks."
        else:
            description = f"No DNS TXT records were found for any domain in the attack surface. "
            description += "DNS TXT records serve critical security functions including SPF email authentication, DKIM key publication, DMARC policy distribution, and domain ownership verification. "
            description += "The complete absence of TXT records indicates that no email authentication, domain verification, or security policies are published in DNS. "
            description += "Without TXT records, the organization cannot implement email spoofing protections, verify domain ownership for third-party services, or publish security contact information. "
            description += "This finding carries a penalty of -50 points due to the zero_entries_fail scoring logic."

            recommendation = "Configure DNS TXT records for all organizational domains, prioritizing SPF, DKIM, and DMARC records for email security. "
            recommendation += "Publish domain verification records required by cloud services and third-party platforms. "
            recommendation += "Consider adding security.txt and CAA records to enhance the overall DNS security posture. "
            recommendation += "Implement DNS monitoring to detect unauthorized TXT record changes or deletions. "
            recommendation += "Priority: Critical - configure core email authentication TXT records within 7 days."

        finding = {
            'Category': 'DNS Health',
            'Tab': dns_type,
            'Priority': severity,
            'Item': f"MISSING {dns_type} - {fail_description}",
            'Description': description,
            'Recommendation': recommendation,
            'Vuln_Type': f'{dns_type}_MISSING',
            'Asset': 'All domains',
            'Plugin_Name': f'{dns_type} Zero Entries Check',
            'Tracking_Status': 'OPEN',
            'FP_Status': 'Verified'
        }
        tab_findings.append(finding)
        group_stats['findings_extracted'] += 1
        all_findings.extend(tab_findings)
        print(f"  {dns_type}: 1 CRITICAL finding (zero_entries_fail)")
        continue

    # ROWS EXIST: Analyze the records for misconfiguration
    for idx, row in enumerate(rows):
        group_stats['total_rows'] += 1
        fp = get_fp(row)
        if fp == 1:
            group_stats['fp_excluded'] += 1
            continue

        tracking = get_tracking(row)
        group_stats['tracking'][tracking] = group_stats['tracking'].get(tracking, 0) + 1
        confidence, visibility, risk = get_cvr_scores(row)
        if confidence is not None:
            group_stats['cvr_scores'].append((confidence, visibility, risk))

        data_field = str(row.get('Data', '')).strip()
        source = str(row.get('Source', '')).strip()
        module = str(row.get('Module', '')).strip()

        severity = 'Medium'

        # Check for specific misconfigurations in SPF/TXT records
        data_lower = data_field.lower()
        if dns_type == 'DNS_SPF':
            if '+all' in data_lower:
                severity = 'Critical'
                misconfig = 'SPF record uses +all (permits all senders)'
            elif '~all' in data_lower:
                severity = 'Medium'
                misconfig = 'SPF record uses ~all (softfail - not enforcing)'
            elif '?all' in data_lower:
                severity = 'High'
                misconfig = 'SPF record uses ?all (neutral - no protection)'
            elif 'include' in data_lower and data_lower.count('include') > 10:
                severity = 'Medium'
                misconfig = 'SPF record has excessive includes (DNS lookup limit risk)'
            else:
                misconfig = 'SPF record present - review for completeness'
                severity = 'Low'

            description = f"{record_name} record analyzed for {source}: {misconfig}. "
            description += f"SPF record content: {data_field[:400]}. "
            description += "SPF records control which servers are authorized to send email for the domain and are fundamental to preventing email spoofing. "
            description += "Misconfigured SPF records can either fail to prevent spoofing or break legitimate email delivery. "
            if confidence is not None and confidence < 50:
                description += f"Note: Confidence score is {confidence}/100 - manual verification of SPF configuration recommended."
            else:
                description += "Regular SPF record audits should be conducted as authorized senders change over time."
        else:
            description = f"{record_name} record analyzed for {source}. "
            description += f"TXT record content: {data_field[:400]}. "
            description += "DNS TXT records serve multiple security functions including email authentication, domain verification, and security policy publication. "
            description += "Each TXT record should be reviewed to ensure it serves a current purpose and does not expose sensitive information. "
            description += "Stale or misconfigured TXT records can reveal infrastructure details or create security gaps."

        recommendation = f"Review the {dns_type} record configuration for {source} to ensure it accurately reflects current infrastructure and security requirements. "
        recommendation += "Validate that email authentication records (SPF/DKIM/DMARC) are correctly configured and enforcing policy. "
        recommendation += "Remove stale or unnecessary TXT records that may expose internal information. "
        recommendation += "Implement DNS change monitoring to detect unauthorized modifications to security-critical records. "
        recommendation += f"Priority: {severity} - address according to organizational DNS management SLAs."

        finding = {
            'Category': 'DNS Health',
            'Tab': dns_type,
            'Priority': severity,
            'Item': f"{dns_type} Record - {source} - {data_field[:60]}",
            'Description': description,
            'Recommendation': recommendation,
            'Vuln_Type': dns_type,
            'Asset': source,
            'Plugin_Name': module,
            'Tracking_Status': tracking,
            'FP_Status': "Verified" if fp == 2 else "Unverified"
        }
        if risk is not None:
            finding['Avg_Risk'] = risk
        tab_findings.append(finding)
        group_stats['findings_extracted'] += 1

    all_findings.extend(tab_findings)
    print(f"  {dns_type}: {len(tab_findings)} findings")
```

```python
        'Tracking_Status': tracking,
        'FP_Status': "Verified" if fp == 2 else "Unverified"
    }
    if risk is not None:
        finding['Avg_Risk'] = risk
    tab_findings.append(finding)
    group_stats['findings_extracted'] += 1

all_findings.extend(tab_findings)
print(f"NETBLOCK_WHOIS: {len(tab_findings)} findings")
```

```python
print("\n=== CONSOLIDATION PHASE ===")

groups = defaultdict(list)
for f in all_findings:
    key = (f['Category'], f['Vuln_Type'], f['Priority'])
    groups[key].append(f)

print(f"Consolidation groups: {len(groups)}")
for key, findings in groups.items():
    if len(findings) > 1:
        print(f"  {key}: {len(findings)} findings")

consolidated_findings = []
os.makedirs('./output/findings_details', exist_ok=True)

for key, group_findings in groups.items():
    if len(group_findings) == 1:
        consolidated_findings.append(group_findings[0])
    else:
        category, vuln_type, priority = key
        assets = [f['Asset'] for f in group_findings]
        unique_assets = list(set([a for a in assets if a and a.lower() not in ['nan', 'none']]))
        asset_count = len(unique_assets)
        instance_count = len(group_findings)

        asset_list = ", ".join([f"({i+1}) {unique_assets[i]}" for i in range(min(5, len(unique_assets)))])
        if len(unique_assets) > 5:
            asset_list += f", and {len(unique_assets)-5} more"

        tracking_counts = defaultdict(int)
        for f in group_findings:
            tracking_counts[f.get('Tracking_Status', 'OPEN')] += 1
        tracking_summary = ", ".join([f"{k}: {v}" for k, v in tracking_counts.items()])

        avg_risk = None
        risk_scores = [f.get('Avg_Risk') for f in group_findings if f.get('Avg_Risk') is not None]
        if risk_scores:
            avg_risk = sum(risk_scores) / len(risk_scores)

        vuln_name = vuln_type.replace('_', ' ').title()
        _lead = f"{vuln_name} was identified on {instance_count} instances across {asset_count} assets: {asset_list}."
        _default_desc = (
            _lead + " This represents a systematic security issue affecting multiple systems "
            "requiring coordinated remediation. The widespread nature indicates common "
            "misconfiguration, shared vulnerable components, or consistent security control gaps."
        )
        _default_rec = (
            f"Implement systematic remediation across all {instance_count} affected instances. "
            "Deploy consistent security controls and monitoring across all affected assets. "
            f"Prioritize remediation based on asset criticality and verify each of the "
            f"{asset_count} affected assets through targeted testing."
        )
        description, recommendation = _enrich(vuln_type, _lead, _default_desc, _default_rec)

        consolidated = {
            'Category': category,
            'Tab': group_findings[0]['Tab'],
            'Priority': priority,
            'Item': f"{vuln_name} - {instance_count} instances across {asset_count} assets",
            'Description': description,
            'Recommendation': recommendation,
            'Vuln_Type': vuln_type,
            'Asset': ", ".join(unique_assets[:5]) if unique_assets else '',
            'Plugin_Name': group_findings[0].get('Plugin_Name', ''),
            'Tracking_Status': max(tracking_counts, key=tracking_counts.get),
        }
        if avg_risk is not None:
            consolidated['Avg_Risk'] = round(avg_risk, 1)
        consolidated_findings.append(consolidated)

        # Create detail file
        import csv as csv_mod
        detail_data = [{'Instance': i+1, 'Asset': f['Asset'],
                        'Details': f['Item'], 'Description': f['Description'][:500],
                        'Tracking_Status': f.get('Tracking_Status', 'OPEN'),
                        'FP_Status': f.get('FP_Status', 'Unverified')}
                       for i, f in enumerate(group_findings)]
        detail_file = f'./output/findings_details/findings_detail_{vuln_type}.csv'
        with open(detail_file, 'w', newline='') as csvf:
            writer = csv_mod.DictWriter(csvf, fieldnames=detail_data[0].keys())
            writer.writeheader()
            writer.writerows(detail_data)
        print(f"  Created: {detail_file}")

print(f"\nConsolidation complete: {len(all_findings)} -> {len(consolidated_findings)} findings")
```
